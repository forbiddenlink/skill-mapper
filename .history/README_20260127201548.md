## ğŸ“ Skill Tree Structure

Skills are organized into **tiers** representing learning progression:

1. **Foundation** - Web Standards, Git, JavaScript, Python
2. **Frontend** - React, TypeScript, Tailwind CSS, State Management
3. **Backend & Data** - Node.js, PostgreSQL, APIs, ORM
4. **AI Engineer** - LLM Integration, RAG Architecture, Vector DBs
5. **Systems** - Advanced topics and specializations

Each skill includes:
- **Title & Description**: What you'll learn
- **Prerequisites**: Skills required before unlocking
- **Resources**: Curated learning materials (videos, articles, courses)
- **Quizzes**: Optional knowledge verification
- **XP Reward**: Points earned upon completion

## ğŸ® How to Use

1. **Start with Available Skills**: Begin with unlocked foundation skills (no prerequisites)
2. **Click a Node**: View skill details, resources, and requirements
3. **Unlock Skills**: Click "Begin Learning" to unlock a skill
4. **Complete Quizzes**: Test your knowledge with interactive challenges
5. **Master Skills**: Mark skills as mastered to unlock dependent skills
6. **Earn Badges**: Complete skill tiers to unlock achievements
7. **Track Progress**: Monitor your XP, level, and streak in the HUD

### Keyboard Shortcuts
- **Arrow Keys**: Navigate between skill nodes
- **Escape**: Close panels/modals
- **Click**: Select and interact with skills

## ğŸ› ï¸ Customization

### Adding New Skills

Edit `src/lib/skill-data.ts`:

```typescript
{
  id: 'my-new-skill',
  type: 'skill',
  data: {
    id: 'my-new-skill',
    title: 'My New Skill',
    description: 'What you will learn',
    tier: 'foundation',
    category: 'frontend',
    status: 'locked',
    prerequisites: ['web-standards'], // IDs of required skills
    xpReward: 150,
    resources: [
      { label: 'Tutorial', url: 'https://...', type: 'course' }
    ],
    quiz: [ /* optional */ ]
  }
}
```

### Creating Badges

Edit `src/lib/badges.ts`:

```typescript
{
  id: 'my-badge',
  label: 'Badge Name',
  description: 'How to earn this badge',
  icon: YourLucideIcon,
  color: 'text-blue-400',
  requirements: ['skill-id-1', 'skill-id-2'] // Skills to master
}
```
